# Notes Template (per batch)

Folder:
`notes/<YYYY-MM-DD>/batch-<NN>/`

Example:
`notes/2026-02-28/batch-01/`

## 1) batch-summary.md
- Date/time
- Role + JD path
- Criteria path + overrides used
- Candidate count
- Ranked results (name, score, decision)
- Clear advances / maybes / no
- Follow-ups needed

## 2) candidate-notes.md
For each candidate:
- Name / ID
- Score + decision
- Hard gates: PASS/FAIL/UNCERTAIN
- Strengths
- Risks
- Detailed analysis (consulting fit, technical fluency, comms, etc.)

## 3) evidence-log.md
For each candidate:
- Resume source files
- LinkedIn URL(s) checked via browser
- Other web sources (if any)
- Explicit facts
- Inferred facts
- Confidence level (high/medium/low)
