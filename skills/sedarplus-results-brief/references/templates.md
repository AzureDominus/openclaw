# Templates - SEDAR+ Results Brief

## A) Search prompt scaffold (external reporting)

Use variants like:

- `{issuer} Q4 {year} results commentary analyst`
- `{issuer} fiscal {year} annual results`
- `{issuer} earnings call transcript Q4 {year}`
- `{issuer} Reuters {month year} results`

Capture per source:

- date
- headline
- URL
- 1-line angle

## B) Comparison memo skeleton

- Local files found:
  - what exists
  - whether relevant to public disclosure
- Official filings found:
  - key rows by date and class
- External reporting found:
  - dated list with angles
- Consistency check:
  - consistent / mixed / conflicting
- Confidence and caveats:
  - blockers, paywalls, inaccessible pages

## C) Deck outline (6 slides)

1. Cover + scope
2. Official filing trail (what was filed and when)
3. Headline metrics from official docs
4. External commentary themes
5. Local vs official vs web comparison
6. Recommended next steps / monitoring points

## D) Team email draft skeleton

Subject: `{issuer} Q{quarter}/FY{year} public filing brief (SEDAR+ validated)`

Body:

- one-line objective
- bullet list of key official findings
- bullet list of external read-through
- list of deliverables and paths
- optional follow-up options

## E) Rerun checklist

- Confirm issuer profile number
- Set filing category + date range
- Capture page 1 + page 2 rows
- Save click-path notes
- Extract metrics from official docs
- Add dated external commentary
- Publish deck + memo + email draft
